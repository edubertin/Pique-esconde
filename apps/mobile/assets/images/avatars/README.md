# Avatars

Coloque aqui os assets finais dos avatares usados pelo app.

## Padrao de arquivos

```txt
avatar-girl-01-face.png
avatar-girl-01-celebrate.png
avatar-girl-02-face.png
avatar-girl-02-celebrate.png
avatar-boy-01-face.png
avatar-boy-01-celebrate.png
avatar-robot-01-face.png
avatar-robot-01-celebrate.png
```

## Regras

- Use PNG com fundo transparente.
- Use imagens quadradas, preferencialmente `1024x1024`.
- A versao `face` deve priorizar rosto grande para lobby e selecao.
- A versao `celebrate` deve mostrar pose comemorando para resultado e card social.
- Nao incluir texto, logo ou fundo de cenario dentro do asset.
