# ADR-004 - No Full Auth in MVP

## Status

Aceita.

## Contexto

O jogo é ocasional, de grupo e precisa começar com pouca fricção. Login completo adiciona barreira e coleta de dados.

## Decisão

Não exigir conta permanente no MVP. Usar sessão temporária por sala com apelido e avatar.

## Consequências

- Entrada mais rápida.
- Menos dados pessoais.
- Menos complexidade inicial.
- Rankings, perfis e histórico ficam fora do MVP.
