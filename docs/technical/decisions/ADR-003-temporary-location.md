# ADR-003 - Temporary Location

## Status

Aceita.

## Contexto

Localização é central para radar e captura, mas o produto não deve parecer monitoramento de pessoas.

## Decisão

Usar localização apenas durante a partida e como dado temporário.

## Consequências

- Menos risco de privacidade.
- Comunicação mais clara para usuários.
- O app deve parar de usar localização ao fim da partida.
- Não deve guardar histórico permanente de rotas.
