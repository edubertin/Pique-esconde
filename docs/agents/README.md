# Agents

Agentes especializados para apoiar o desenvolvimento do Pique Esconde.

Cada agente descreve um papel, contexto, responsabilidades, arquivos que deve conhecer e formato esperado de resposta. Esses documentos podem ser usados como prompt base em sessões futuras com IA ou como briefing para colaboradores humanos.

## Agentes Disponíveis

- [Pique Esconde Game UI Designer](game-ui-designer.md)

## Como Usar

1. Leia o arquivo do agente.
2. Atualize o contexto se houver novas decisões de produto.
3. Peça uma entrega específica, como revisão de tela, design system, redesign do lobby ou direção visual do radar.
4. Valide se as recomendações respeitam as specs, ADRs e regras de privacidade.
