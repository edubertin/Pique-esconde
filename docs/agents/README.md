# Agents

Agentes especializados para apoiar o desenvolvimento do Pique Esconde.

Cada agente descreve um papel, contexto, responsabilidades, arquivos que deve conhecer e formato esperado de resposta. Esses documentos podem ser usados como prompt base em sessoes futuras com IA ou como briefing para colaboradores humanos.

## Agentes Disponiveis

- [Pique Esconde Game UI Designer](game-ui-designer.md)
- [Pique Esconde Supabase Realtime GPS Architect](supabase-realtime-gps-architect.md)

## Como Usar

1. Leia o arquivo do agente.
2. Atualize o contexto se houver novas decisoes de produto.
3. Peca uma entrega especifica, como revisao de tela, design system, redesign do lobby, direcao visual do radar, auditoria de GPS/Supabase ou desenho de ferramenta de debug.
4. Valide se as recomendacoes respeitam as specs, ADRs e regras de privacidade.
